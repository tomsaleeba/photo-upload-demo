Image upload demo

This tool demonstrates a bandwith saving method of uploading images. You can add images by dragging and dropping them to the area below. Images will then be resized to thumbnail dimensions and uploaded to a server. The server will then automatically classify the images and tag images that contain a feature of interest. You can then upload the full size version of only these relevant images. Other images, with nothing in them, are ignored.

Note: the autoclassification on the server-side isn't currently implemented, it's stubbed for this demo.
